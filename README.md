# Vidly from Mosh's Node JS course

```
npm install
set vidly_jwtPrivateKey=mySecureKey
node index.js
```

## WebImblaze tests

Install WebImblaze, WebImblaze-FrameWork, then add `wif.pl` to path.

```
wif.pl test.test
```

## Deployment

Install heroku cli
```
https://devcenter.heroku.com/articles/heroku-cli
```

Login
```
heroku login
```
